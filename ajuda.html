Guia de Uso do Sistema

Siga o passo a passo abaixo para gerar o orçamento de recebimento corretamente.

Finalização (Gustavo)

  • Insira seus dados de produção (Gustavo).
  • Clique em Calcular. O sistema buscará os dados das páginas anteriores e fará a soma total.

Cadastro do Jorge

  • Na segunda tela, repita o processo com os dados do Jorge.
  • Clique em Próximo. Não se preocupe, os dados da Fernanda ainda estão guardados!

Cadastro da Fernanda

  • Insira as quantidades de buchas (G, P, V) produzidas pela Fernanda.
  • Clique no botão Próximo. O sistema salvará esses dados automaticamente na memória do seu navegador.

Resultado e WhatsApp

Um pop-up aparecerá com o resumo detalhado de quanto cada um tem a receber.

Dica: Ao clicar em "Enviar Orçamento", o sistema prepara uma mensagem para o WhatsApp com todos os valores organizados.
